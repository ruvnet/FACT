### MCP Server Integration
```bash
# Add to Claude Desktop
claude mcp add fact-mcp npx @fact/mcp-server

# Or run standalone
npx @fact/mcp-server
```

### WASM Module Usage
```javascript
// In Node.js
import { WasmLoader } from 'fact-cli';
const loader = new WasmLoader();
await loader.initialize();
const result = loader.callFunction('fact-core', 'process_query', queryString);

// In Browser
import init, { FastCache, QueryProcessor } from './pkg/fact_wasm_core.js';
await init();
const cache = new FastCache(1000);
const processor = new QueryProcessor();
```
